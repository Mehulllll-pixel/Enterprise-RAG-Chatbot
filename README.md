# Enterprise On-Premise RAG Chatbot

This is the workspace root for the Enterprise On-Premise Retrieval-Augmented Generation (RAG) Chatbot.

## Subdirectories

- `/backend`: FastAPI, SQLAlchemy, Alembic, FAISS, and LangChain/Ollama pipeline.
- `/frontend`: React 19, TypeScript, Vite, Tailwind CSS, and shadcn/ui.
